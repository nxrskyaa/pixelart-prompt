02PRO TIPS
LOCK PERSPECTIVE

Always say "top-down 3/4 view" + "not side-scroll". Models default to side-view if you say "platformer".

MAGENTA KEY BG

Flat #FF00FF background = 1-click removal later in any editor.

CAP THE PALETTE

"max ~16 colors" forces real pixel shading instead of smooth gradients.

KILL ANTI-ALIASING

Repeat "no anti-aliasing, no blur, hard edges" — the #1 thing models forget.

GRID + FRAMES

State exact rows×cols ("4 rows of 4 frames") so the sheet comes back aligned.

IDENTITY FIRST

Tell it to "preserve face, hair, outfit, colors, proportions" for consistency.
