02 PRO TIPS — GET CLEANER SPRITES
🎯
LOCK THE PERSPECTIVE

Always say "top-down 3/4 view" + "no side-scroll". Models default to side-view if you mention "platformer".

🟪
MAGENTA KEY BG

Flat #FF00FF background = easy 1-click removal later in any editor or remove.bg.

🔢
CAP THE PALETTE

"max ~16 colors" forces real pixel-art shading instead of smooth painted gradients.

🚫
KILL ANTI-ALIASING

Repeat "no anti-aliasing, no blur, hard edges, crisp pixels" — it's the #1 thing models forget.

📐
GRID + FRAMES

State exact rows×cols ("4 rows of 4 frames") so the spritesheet comes back evenly aligned.

🧍
IDENTITY FIRST

Tell it to "preserve face, hair, outfit, colors, proportions" so the character stays consistent across frames.
