# PIXELFORGE — Pixel Art Prompt Vault

A copy-paste prompt vault for generating **Stardew Valley / Play Kintara** style **top-down 3/4 perspective** pixel art sprites & game assets with AI image models (ChatGPT, DALL·E, etc.).

🎮 **Live:** [pixelart-prompt.vercel.app](https://pixelart-prompt.vercel.app)

## What's inside

5 battle-tested, tuned prompts:

1. **Character Sprite — From Reference** — single idle + 4-direction turnaround
2. **Walk Cycle Spritesheet** — full 4-direction (down/up/left/right) walk animation, 16 frames
3. **Full Action Set** — idle + walk + run, 12-16 frame multi-state sheet
4. **Building / Structure Asset** — cozy top-down building (Eastward + Stardew vibe)
5. **Environment Props / Tileset Kit** — matching trees, rocks, crops, fences

Plus: recommended cozy muted palette, pro tips for cleaner sprites, and one-click copy.

## Key fix vs. raw prompts

The original prompts mixed **side-scroll platformer / side-view** with **Stardew Valley** references — those contradict. Stardew & Play Kintara are **top-down 3/4**. Every prompt here locks the correct perspective and explicitly says "NOT side-scroll".

## Stack

Pure static HTML/CSS/JS. No build step. Press Start 2P / VT323 / Silkscreen fonts, CRT scanline overlay, pixel-perfect UI.
